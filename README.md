# Student-Mental-Health-Analytics-Dashboard
---

## 🔍 Project Overview
This project presents an interactive **Tableau dashboard** focused on analyzing student mental health trends using real-world survey data.  
It highlights how factors like academic pressure, sleep patterns, and lifestyle habits impact mental well-being.

---

## 🎯 Objectives
- Analyze student mental health data to uncover patterns  
- Identify key contributors to stress, anxiety, and depression  
- Build an interactive and user-friendly dashboard  
- Enable data-driven insights for better decision-making  

---

## 🛠️ Tools & Technologies
- 📊 **Tableau** – Data visualization & dashboard creation  
- 📁 **CSV Dataset** – Student mental health survey data  

---

## 📈 Key Features
- Interactive filters for dynamic exploration  
- KPI indicators for quick insights  
- Visualizations: bar charts, pie charts, trend graphs  
- Correlation analysis between lifestyle and mental health
